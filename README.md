## Agentic Web using WebMCP

This repository holds the code for the demos for the talk "Agentic Web using WebMCP". Two projects are included:

- A Flutter project, which can be found in the `WebMCPDemo/flutter-sample` folder.
- A React project, which can be found in the `WebMCPDemo/react-sample` folder.
